SOF platform provides backend API with management system and real time tracking of processes. Also client sides can be built (both, customer of the service and performer) depending on the business processes.

 

Similar products that can be built on SOF platform:

CarOnFly, Uber, Gett, Lyft, Postmates, YouDo, etc.

ServicesOnFly

GOODSONFLY

Solutions that can be provided on this platform:

Selling or Renting Real Estate, Cars, Boats, any other stuff, even from your neighbors. Also all this is linked to the geolocation.

GoodsOF platform provides backend API with management system and possibility to upload and manage the database from the frontend (web or client). Also client side can be built for customer or seller.

Similar products that can be built on GoodsOF platform: Trulia, Getaround, GetMyBoat, OfferUp, etc.

GETONFLY

Solutions that can be provided on this platform: aggregation of proposals on food/grocery/clothes/anything else delivery from restaurants/shops, etc.

GetOF platform provides backend API with management system and possibility to upload and manage the database from the frontend (web or client). Also client side can be built for end-users.

Similar products that can be built on GetOF platform:  Instacart, Yelp Eat24, UberEATS, GrubHub, DeliveryClub, JackThreads, Mama, etc.

Where else can be used: as a menu app for restaurants with instant data transfer to a kitchen.

ACTIVITIESONFLY

Solutions that can be provided on this platform: any content providing from business/people who have something to say and it can be sold. Any personal trainers (in sports or anywhere else), chiefs, dance studios, martial arts clubs, etc. Also end-customers can track their record and interact with content provider.

AOF platform provides backend API with management system and possibility to upload and manage content from the frontend (web or client). Also client side can be built for end-users.

Similar products that can be built on AOF platform: PersonalTrainer, Recipes by Jamie Oliver, Ski4U, etc.

Where else can be used: as an app for sport clubs where they can provide loyalty program and training program/schedule for their customers.

FINANCEONFLY

Solutions that can be provided on this platform: peer-to-peer loans; gathering money from users for any purpose (charity, event, project, etc.) with/without following transfer; payment system.

FOF platform provides backend API with management and risk-management systems (depends on the business processes) and possibility of integration with the frontend (web or client) for specific purposes. Also client side can be built for end-users.

Similar products that can be built on FOF platform: LendingClub, BeGoodFriend, P2PLending, Wallet One.
